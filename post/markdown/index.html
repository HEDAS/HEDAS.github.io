markdown语法

常用语法

1. 标题（#号后面要有空格）

# h1标题
## h2标题
### h3标题
#### h4标题
##### h5标题
###### h6标题

2. 分级标题（=、-最少可写一个）

一级标题
=====================
二级标题
---------------------

3. 引用（要写空行隔开）

>这是一级引用

>>这是二级引用

>>>这是三级引用

>这是一级引用

4. [TOC]（博客系统不支持）

[TOC]

5. 标记（多行代码会合成一行）

行首`for i in range(10):
    print('Hello,world!')`行尾

6. 代码块

方法一：

```
for i in range(10):
    print("Hello,world!")
```

方法二：

#用tab缩进
    for i in range(10):
    print("Hello,world!")
